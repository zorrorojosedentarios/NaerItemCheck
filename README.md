# NaerItemCheck

Addon para World of Warcraft 3.3.5a (WotLK) diseñado específicamente para el servidor **NaerZone**. 
Permite verificar rápidamente si un jugador objetivo posee objetos específicos de la banda **Icecrown Citadel (ICC)** mediante comandos automáticos al servidor.

## Características principales
- **Base de datos de ICC:** Incluye todos los jefes y objetos de las dificultades 10N, 10H, 25N y 25H.
- **Interfaz Premium:** Ventana de 600x500 con iconos detallados y estados por colores.
- **Editor de IDs integrado:** Modifica la base de datos en tiempo real desde el juego (botón de engranaje).
- **Persistencia:** Los cambios realizados se guardan por cuenta/personaje.
- **Acceso rápido:** Botón en el minimapa y soporte para la tecla Escape.

## Comandos
- `/nic` - Abre o cierra la interfaz principal.

## Instalación
1. Descarga el addon.
2. Extrae la carpeta `NaerItemCheck` en `Interface/Addons/`.
3. Reinicia el juego o recarga la interfaz (`/reload`).

## Estados
- **Gris:** Sin datos.
- **Celeste:** Consultando al servidor...
- **Verde:** El jugador tiene el objeto.
- **Rojo:** El jugador NO tiene el objeto.
